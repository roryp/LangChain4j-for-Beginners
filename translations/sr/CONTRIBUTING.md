# Contributing

Хвала вам што сте заинтересовани да допринесете! Добродошли су захтеви за повлачење и побољшања овог курса. Молимо вас да пратите ове смернице када подносите измене:

1. **Нема тајни у комитима.** Никада не комитујте API кључеве или било који поверљив материјал. Користите локални `.env` за своје тестирање.
2. **Држите измене фокусираним.** Мали, ограничени захтеви за повлачење су лакши за преглед и одржавање. Ако исправљате више проблема, молимо вас да их пошаљете одвојено.
3. **Додајте тестове где је могуће.** Јединични или интеграциони тестови помажу у откривању регресија. За Spring Boot модуле, укључите тестове у фасциклу `src/test/java`.
4. **Пратите конвенције стила кода.** Користите постојећу структуру пројекта као смерницу. Ослањамо се на Maven за управљање зависностима и Spring Boot Maven плагин за паковање апликација.

Хвала вам што помажете да овај курс буде бољи!
