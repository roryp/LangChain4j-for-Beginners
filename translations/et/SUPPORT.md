# Tugi

## Kuidas esitada probleeme ja saada abi  

See projekt kasutab vigade ja funktsioonisoovide jälgimiseks GitHub Issues. Palun otsige olemasolevaid 
probleeme enne uute probleemide esitamist, et vältida dubleeringuid. Uute probleemide puhul esitage oma viga või 
funktsioonisoov uue probleemina.
