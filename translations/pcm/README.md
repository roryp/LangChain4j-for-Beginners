## Learning Path

> **Quick Start**

1. Fork dis repository go your GitHub account
2. Click **Code** ‚Üí **Codespaces** tab ‚Üí **...** ‚Üí **New with options...**
3. Use di defaults ‚Äì dis go select di Development container wey dem create for dis course
4. Click **Create codespace**
5. Wait 5-10 minutes make di environment ready
6. Jump sharp go [Quick Start](./00-quick-start/README.md) to start!

> **Prefer to Clone Locally?**
>
> Dis repository get more dan 50 language translations wey dey increase di download size well well. To clone without di translations, use sparse checkout:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/LangChain4j-for-Beginners.git

After you finish di modules, explore di [Testing Guide](docs/TESTING.md) to see how LangChain4j testing concepts dey work.

> **Note:** Dis training dey use both GitHub Models and Azure OpenAI. Di [Quick Start](00-quick-start/README.md) module dey use GitHub Models (no need subscription for Azure), but modules 1-5 dey use Azure OpenAI.


## Learning with GitHub Copilot

To start code quick quick, open dis project for GitHub Codespace or your local IDE wit di provided devcontainer. Di devcontainer wey dem use for dis course get pre-configured GitHub Copilot for AI paired programming.

Every code example get suggested questions wey you fit ask GitHub Copilot to make you understand better. Look for di üí°/ü§ñ prompts inside:

- **Java file headers** - Questions wey specific to each example
- **Module READMEs** - Prompts to explore after code examples

**How to use:** Open any code file and ask Copilot di suggested questions. E sabi whole code well and fit explain, add more, or suggest different options.

You want learn more? Check out [Copilot for AI Paired Programming](https://aka.ms/GitHubCopilotAI).
