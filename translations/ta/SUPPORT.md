# ஆதரவு

## பிரச்சனைகளை பதிவு செய்வது மற்றும் உதவி பெறுவது எப்படி  

இந்த திட்டம் பிழைகள் மற்றும் அம்ச கோரிக்கைகளை கண்காணிக்க GitHub Issues ஐ பயன்படுத்துகிறது. 
புதிய பிரச்சனைகளை பதிவு செய்வதற்கு முன், நகல்களைத் தவிர்க்க ஏற்கனவே உள்ள பிரச்சனைகளை 
தேடவும். புதிய பிரச்சனைகளுக்கு, உங்கள் பிழை அல்லது அம்ச கோரிக்கையை புதிய பிரச்சனையாக பதிவு செய்யவும்.
