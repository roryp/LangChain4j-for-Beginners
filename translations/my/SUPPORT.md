# Support

## ပြဿနာများ တင်သွင်းခြင်းနှင့် အကူအညီရယူနည်း  

ဤပရောဂျက်သည် GitHub Issues ကို အသုံးပြု၍ အမှားများနှင့် လုပ်ဆောင်ချက်တောင်းဆိုမှုများကို စောင့်ကြည့်သည်။ မတူညီသော ပြဿနာများ မဖြစ်ပေါ်စေရန် အရင်ရှိပြီးသား ပြဿနာများကို ရှာဖွေကြည့်ပါ။ ပြဿနာအသစ်များအတွက် သင့်အမှား သို့မဟုတ် လုပ်ဆောင်ချက်တောင်းဆိုမှုကို Issue အသစ်အဖြစ် တင်ပြပါ။

## Microsoft Support Policy  

ဤပရောဂျက်အတွက် အထောက်အပံ့သည် အထက်ဖော်ပြထားသော အရင်းအမြစ်များအထိသာ ကန့်သတ်ထားသည်။
