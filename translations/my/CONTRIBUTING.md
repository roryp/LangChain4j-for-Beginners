# Contributing

သင်၏ အထောက်အပံ့ပေးမှုအတွက် ကျေးဇူးတင်ပါသည်! ဤသင်တန်းအတွက် pull requests နှင့် တိုးတက်မှုများကို ကြိုဆိုပါသည်။ ပြင်ဆင်မှုများတင်သွင်းရာတွင် အောက်ပါ လမ်းညွှန်ချက်များကို လိုက်နာပါ။

1. **commit များတွင် လျှို့ဝှက်ချက် မပါရှိစေရန်။** API key များ သို့မဟုတ် လျှို့ဝှက်အချက်အလက် များကို မတင်သွင်းပါနှင့်။ သင်၏ ကိုယ်ပိုင် စမ်းသပ်မှုအတွက် `.env` ဖိုင်ကို အသုံးပြုပါ။
2. **ပြင်ဆင်မှုများကို အာရုံစိုက်ထားပါ။** သေးငယ်ပြီး အကွာအဝေးသတ်မှတ်ထားသော pull requests များသည် ပြန်လည်သုံးသပ်ရန်နှင့် ထိန်းသိမ်းရန် ပိုမိုလွယ်ကူသည်။ ပြဿနာများစွာကို ပြင်ဆင်နေပါက သီးခြားတင်သွင်းပါ။
3. **ဖြစ်နိုင်သမျှ စမ်းသပ်မှုများ ထည့်သွင်းပါ။** ယူနစ် သို့မဟုတ် ပေါင်းစပ် စမ်းသပ်မှုများသည် ပြန်လည်ဆိုးရွားမှုများကို ဖမ်းဆီးရာတွင် အထောက်အကူဖြစ်သည်။ Spring Boot modules များအတွက် `src/test/java` ဖိုလ်ဒါအောက်တွင် စမ်းသပ်မှုများ ထည့်သွင်းပါ။
4. **ကုဒ်စတိုင် စည်းမျဉ်းများကို လိုက်နာပါ။** ရှိပြီးသား ပရောဂျက် ဖွဲ့စည်းမှုကို လမ်းညွှန်အဖြစ် အသုံးပြုပါ။ ကျွန်ုပ်တို့သည် Maven ကို အားထား၍ အားထားပြီး Spring Boot Maven plugin ကို အသုံးပြုကာ အက်ပလီကေးရှင်းများကို ထုပ်ပိုးသည်။

ဤသင်တန်းကို ပိုမိုကောင်းမွန်စေရန် အထောက်အကူပြုမှုအတွက် ကျေးဇူးတင်ပါသည်!
