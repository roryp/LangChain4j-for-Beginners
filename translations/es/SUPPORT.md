# Soporte

## Cómo presentar problemas y obtener ayuda  

Este proyecto utiliza GitHub Issues para rastrear errores y solicitudes de funciones. Por favor, busque en los problemas existentes antes de presentar nuevos problemas para evitar duplicados. Para nuevos problemas, presente su error o solicitud de función como un nuevo Issue.

## Política de soporte de Microsoft  

El soporte para este proyecto se limita a los recursos listados arriba.
