# Contributing

Hvala vam na interesu za doprinos! Dobrodošli su pull zahtjevi i poboljšanja ovog tečaja. Molimo vas da slijedite ove smjernice prilikom slanja promjena:

1. **Nema tajni u commitovima.** Nikada ne commitajte API ključeve ili bilo kakav povjerljiv materijal. Koristite lokalni `.env` za vlastito testiranje.  
2. **Održavajte promjene fokusiranima.** Mali, ograničeni pull zahtjevi lakše se pregledavaju i održavaju. Ako popravljate više problema, molimo da ih pošaljete zasebno.  
3. **Dodajte testove gdje je moguće.** Jedinični ili integracijski testovi pomažu u otkrivanju regresija. Za Spring Boot module, uključite testove u mapu `src/test/java`.  
4. **Slijedite konvencije stila koda.** Koristite postojeću strukturu projekta kao vodič. Oslanjamo se na Maven za upravljanje ovisnostima i Spring Boot Maven plugin za pakiranje aplikacija.

Hvala vam što pomažete da ovaj tečaj bude bolji!
