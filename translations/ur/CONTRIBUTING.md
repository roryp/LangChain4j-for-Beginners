# تعاون

تعاون میں دلچسپی لینے کا شکریہ! ہم اس کورس میں پل ریکویسٹ اور بہتریوں کا خیرمقدم کرتے ہیں۔ براہ کرم تبدیلیاں جمع کراتے وقت ان ہدایات پر عمل کریں:

1. **کمیٹس میں راز نہ رکھیں۔** کبھی بھی API کیز یا کوئی بھی خفیہ مواد کمیٹ نہ کریں۔ اپنی جانچ کے لیے مقامی `.env` استعمال کریں۔
2. **تبدیلیوں کو مرکوز رکھیں۔** چھوٹے، محدود پل ریکویسٹز کا جائزہ لینا اور برقرار رکھنا آسان ہوتا ہے۔ اگر آپ متعدد مسائل کو ٹھیک کر رہے ہیں، تو براہ کرم انہیں الگ الگ جمع کروائیں۔
3. **جہاں ممکن ہو ٹیسٹ شامل کریں۔** یونٹ یا انٹیگریشن ٹیسٹ ریگریشنز کو پکڑنے میں مدد دیتے ہیں۔ Spring Boot ماڈیولز کے لیے، `src/test/java` فولڈر کے تحت ٹیسٹ شامل کریں۔
4. **کوڈ اسٹائل کنونشنز کی پیروی کریں۔** موجودہ پروجیکٹ ڈھانچے کو رہنما کے طور پر استعمال کریں۔ ہم Maven پر انحصار کرتے ہیں تاکہ dependencies کو منظم کیا جا سکے اور Spring Boot Maven پلگ ان کا استعمال کرتے ہیں تاکہ ایپلیکیشنز کو پیک کیا جا سکے۔

اس کورس کو بہتر بنانے میں مدد کرنے کا شکریہ!
