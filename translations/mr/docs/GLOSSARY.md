## मूल संकल्पना

**AI एजंट** - स्वायत्तपणे विचार करणारा आणि कृती करणारा AI वापरणारा प्रणाली. [मॉड्यूल 04](../04-tools/README.md)

**चेन** - ऑपरेशन्सची अशी साखळी जिथे आउटपुट पुढील टप्प्यासाठी इनपुट म्हणून वापरला जातो.

**चंकिंग** - दस्तऐवज लहान तुकड्यांमध्ये विभागणे. सामान्यतः: 300-500 टोकन्स ओव्हरलॅपसह. [मॉड्यूल 03](../03-rag/README.md)

**संदर्भ विंडो** - मॉडेल प्रक्रिया करू शकणारे कमाल टोकन्स. GPT-5: 400K टोकन्स.

**एम्बेडिंग्ज** - मजकूराच्या अर्थाचे संख्यात्मक वेक्टर. [मॉड्यूल 03](../03-rag/README.md)

**फंक्शन कॉलिंग** - मॉडेल बाह्य फंक्शन्स कॉल करण्यासाठी संरचित विनंत्या तयार करते. [मॉड्यूल 04](../04-tools/README.md)

**हॅलुसिनेशन** - जेव्हा मॉडेल चुकीची पण संभाव्य माहिती तयार करतात.

**प्रॉम्प्ट** - भाषा मॉडेलसाठी मजकूर इनपुट. [मॉड्यूल 02](../02-prompt-engineering/README.md)

**सामान्य अर्थाने शोध** - कीवर्डऐवजी एम्बेडिंग्ज वापरून अर्थानुसार शोध. [मॉड्यूल 03](../03-rag/README.md)

**स्टेटफुल विरुद्ध स्टेटलेस** - स्टेटलेस: स्मृती नाही. स्टेटफुल: संभाषण इतिहास राखतो. [मॉड्यूल 01](../01-introduction/README.md)

**टोकन्स** - मॉडेल प्रक्रिया करणारे मूलभूत मजकूर घटक. खर्च आणि मर्यादा यावर परिणाम होतो. [मॉड्यूल 01](../01-introduction/README.md)

**टूल चेनिंग** - टूल्सची सलग अंमलबजावणी जिथे आउटपुट पुढील कॉलसाठी माहिती देते. [मॉड्यूल 04](../04-tools/README.md)

## LangChain4j घटक

**AiServices** - प्रकार-सुरक्षित AI सेवा इंटरफेस तयार करतो.

**OpenAiOfficialChatModel** - OpenAI आणि Azure OpenAI मॉडेलसाठी एकत्रित क्लायंट.

**OpenAiOfficialEmbeddingModel** - OpenAI अधिकृत क्लायंट वापरून एम्बेडिंग्ज तयार करतो (OpenAI आणि Azure OpenAI दोन्हीसाठी समर्थन).

**ChatModel** - भाषा मॉडेलसाठी मुख्य इंटरफेस.

**ChatMemory** - संभाषण इतिहास राखतो.

**ContentRetriever** - RAG साठी संबंधित दस्तऐवज चंक शोधतो.

**DocumentSplitter** - दस्तऐवज चंकमध्ये विभागतो.

**EmbeddingModel** - मजकूर संख्यात्मक वेक्टरमध्ये रूपांतरित करतो.

**EmbeddingStore** - एम्बेडिंग्ज संग्रहित आणि पुनर्प्राप्त करतो.

**MessageWindowChatMemory** - अलीकडील संदेशांचा स्लायडिंग विंडो राखतो.

**PromptTemplate** - `{{variable}}` प्लेसहोल्डर्ससह पुनर्वापरयोग्य प्रॉम्प्ट तयार करतो.

**TextSegment** - मेटाडेटासह मजकूराचा तुकडा. RAG मध्ये वापरला जातो.

**ToolExecutionRequest** - टूल अंमलबजावणी विनंतीचे प्रतिनिधित्व करतो.

**UserMessage / AiMessage / SystemMessage** - संभाषण संदेश प्रकार.

## AI/ML संकल्पना

**फ्यू-शॉट लर्निंग** - प्रॉम्प्टमध्ये उदाहरणे देणे. [मॉड्यूल 02](../02-prompt-engineering/README.md)

**लार्ज लँग्वेज मॉडेल (LLM)** - प्रचंड मजकूर डेटावर प्रशिक्षित AI मॉडेल्स.

**रिझनिंग एफर्ट** - GPT-5 पॅरामीटर जो विचार करण्याची खोली नियंत्रित करतो. [मॉड्यूल 02](../02-prompt-engineering/README.md)

**टेम्परेचर** - आउटपुटची अनियमितता नियंत्रित करते. कमी=निश्चित, जास्त=सर्जनशील.

**व्हेक्टर डेटाबेस** - एम्बेडिंग्जसाठी विशेष डेटाबेस. [मॉड्यूल 03](../03-rag/README.md)

**झीरो-शॉट लर्निंग** - उदाहरणांशिवाय कार्य करणे. [मॉड्यूल 02](../02-prompt-engineering/README.md)

## प्रॉम्प्ट अभियांत्रिकी - [मॉड्यूल 02](../02-prompt-engineering/README.md)

**चेन-ऑफ-थॉट** - अधिक अचूकतेसाठी टप्प्याटप्प्याने विचार.

**बाधित आउटपुट** - विशिष्ट स्वरूप किंवा रचना लागू करणे.

**उच्च उत्सुकता** - GPT-5 नमुना सखोल विचारासाठी.

**कमी उत्सुकता** - GPT-5 नमुना जलद उत्तरांसाठी.

**मल्टी-टर्न संभाषण** - विनिमयांमध्ये संदर्भ राखणे.

**भूमिका-आधारित प्रॉम्प्टिंग** - सिस्टम संदेशांद्वारे मॉडेल व्यक्तिमत्व सेट करणे.

**स्व-परावर्तन** - मॉडेल स्वतःचे आउटपुट मूल्यांकन आणि सुधारणा करते.

**संरचित विश्लेषण** - निश्चित मूल्यांकन चौकट.

**कार्य अंमलबजावणी नमुना** - योजना → अंमलबजावणी → सारांश.

## RAG (रिट्रीव्हल-अगमेंटेड जनरेशन) - [मॉड्यूल 03](../03-rag/README.md)

**दस्तऐवज प्रक्रिया पाइपलाइन** - लोड → चंक → एम्बेड → संग्रहित करा.

**इन-मेमरी एम्बेडिंग स्टोअर** - चाचणीसाठी अस्थायी संग्रहण.

**RAG** - प्रतिसादांना आधार देण्यासाठी रिट्रीव्हल आणि जनरेशन एकत्र करते.

**समानता गुणांक** - अर्थानुसार समानतेचे मापन (0-1).

**स्रोत संदर्भ** - पुनर्प्राप्त केलेल्या सामग्रीबद्दल मेटाडेटा.

## एजंट्स आणि साधने - [मॉड्यूल 04](../04-tools/README.md)

**@Tool अ‍ॅनोटेशन** - Java पद्धतींना AI-कॉल करण्यायोग्य साधने म्हणून चिन्हांकित करते.

**ReAct नमुना** - विचार करा → कृती करा → निरीक्षण करा → पुनरावृत्ती करा.

**सत्र व्यवस्थापन** - वेगवेगळ्या वापरकर्त्यांसाठी वेगळे संदर्भ.

**टूल** - AI एजंट कॉल करू शकणारी फंक्शन.

**टूल वर्णन** - टूलचा उद्देश आणि पॅरामीटर्सचे दस्तऐवजीकरण.

## मॉडेल संदर्भ प्रोटोकॉल (MCP) - [मॉड्यूल 05](../05-mcp/README.md)

**Docker ट्रान्सपोर्ट** - Docker कंटेनरमधील MCP सर्व्हर.

**MCP** - AI अ‍ॅप्सना बाह्य साधनांशी जोडण्यासाठी मानक.

**MCP क्लायंट** - MCP सर्व्हरशी जोडणारे अ‍ॅप्लिकेशन.

**MCP सर्व्हर** - MCP द्वारे साधने उपलब्ध करून देणारी सेवा.

**सर्व्हर-सेंट इव्हेंट्स (SSE)** - HTTP वरून सर्व्हर-टू-क्लायंट स्ट्रीमिंग.

**Stdio ट्रान्सपोर्ट** - stdin/stdout द्वारे उपप्रक्रिया म्हणून सर्व्हर.

**स्ट्रीम करण्यायोग्य HTTP ट्रान्सपोर्ट** - रिअल-टाइम संवादासाठी SSE सह HTTP.

**टूल डिस्कव्हरी** - क्लायंट उपलब्ध साधनांसाठी सर्व्हरला क्वेरी करतो.
