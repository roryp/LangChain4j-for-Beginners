# समर्थन

## समस्या कशी नोंदवायची आणि मदत कशी घ्यायची  

हा प्रकल्प बग आणि वैशिष्ट्य विनंत्या ट्रॅक करण्यासाठी GitHub Issues वापरतो. कृपया नवीन समस्या नोंदवण्यापूर्वी विद्यमान 
समस्यांचा शोध घ्या जेणेकरून डुप्लिकेट टाळता येतील. नवीन समस्यांसाठी, आपली बग किंवा 
वैशिष्ट्य विनंती नवीन Issue म्हणून नोंदवा.
