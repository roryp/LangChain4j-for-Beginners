# योगदान देणे

योगदान देण्यास आपले स्वागत आहे! आम्ही या कोर्समध्ये पुल विनंत्या आणि सुधारणा स्वीकारतो. कृपया बदल सादर करताना या मार्गदर्शक तत्त्वांचे पालन करा:

1. **कमिट्समध्ये कोणतेही रहस्य नसावे.** API की किंवा कोणतीही गोपनीय सामग्री कधीही कमिट करू नका. आपल्या स्वतःच्या चाचणीसाठी स्थानिक `.env` वापरा.
2. **बदल लक्ष केंद्रित ठेवा.** लहान, मर्यादित पुल विनंत्या पुनरावलोकन आणि देखभाल करणे सोपे असतात. आपण अनेक समस्या दुरुस्त करत असाल तर कृपया त्यांना वेगळे सादर करा.
3. **जिथे शक्य असेल तिथे चाचण्या जोडा.** युनिट किंवा इंटिग्रेशन चाचण्या पुनरावृत्ती पकडण्यास मदत करतात. स्प्रिंग बूट मॉड्यूलसाठी, `src/test/java` फोल्डरखाली चाचण्या समाविष्ट करा.
4. **कोड शैलीच्या नियमांचे पालन करा.** विद्यमान प्रकल्प संरचनेचा मार्गदर्शक म्हणून वापर करा. आम्ही Maven वर अवलंबून आहोत जे अवलंबित्वे व्यवस्थापित करते आणि स्प्रिंग बूट Maven प्लगइन वापरून अनुप्रयोग पॅकेज करतो.

हा कोर्स अधिक चांगला बनवण्यासाठी मदत केल्याबद्दल धन्यवाद!
