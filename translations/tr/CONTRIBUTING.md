# Katkıda Bulunma

Katkıda bulunmaya ilginiz için teşekkürler! Bu kursa yapılan çekme isteklerini ve iyileştirmeleri memnuniyetle karşılıyoruz. Değişiklik gönderirken lütfen şu kurallara uyun:

1. **Commitlerde gizli bilgi olmasın.** API anahtarları veya herhangi bir gizli materyali asla commit etmeyin. Kendi testleriniz için yerel bir `.env` dosyası kullanın.
2. **Değişiklikleri odaklı tutun.** Küçük, kapsamlı çekme istekleri incelemesi ve bakımı daha kolaydır. Birden fazla sorunu düzeltiyorsanız, lütfen bunları ayrı ayrı gönderin.
3. **Mümkünse test ekleyin.** Birim veya entegrasyon testleri gerilemeleri yakalamaya yardımcı olur. Spring Boot modülleri için, testleri `src/test/java` klasörü altında ekleyin.
4. **Kod stili kurallarına uyun.** Mevcut proje yapısını rehber olarak kullanın. Bağımlılıkları yönetmek için Maven'e ve uygulamaları paketlemek için Spring Boot Maven eklentisine güveniyoruz.

Bu kursu daha iyi hale getirmemize yardımcı olduğunuz için teşekkürler!
