# Prispevanje

Hvala za vaše zanimanje za prispevanje! Veselimo se pull requestov in izboljšav tega tečaja. Prosimo, upoštevajte naslednja navodila pri pošiljanju sprememb:

1. **Brez skrivnosti v commitih.** Nikoli ne vključujte API ključev ali kakršnih koli zaupnih podatkov. Za lastno testiranje uporabite lokalno datoteko `.env`.
2. **Ohranjajte spremembe osredotočene.** Majhni, omejeni pull requesti so lažji za pregled in vzdrževanje. Če odpravljate več težav, jih prosimo pošljite ločeno.
3. **Dodajte teste, kjer je mogoče.** Enotni ali integracijski testi pomagajo ujeti regresije. Za Spring Boot module vključite teste v mapo `src/test/java`.
4. **Sledite konvencijam kode.** Uporabite obstoječo strukturo projekta kot vodilo. Za upravljanje odvisnosti uporabljamo Maven in Spring Boot Maven plugin za pakiranje aplikacij.

Hvala, ker pomagate izboljšati ta tečaj!
