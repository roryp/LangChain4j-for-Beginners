# समर्थन

## कसरी मुद्दा दर्ता गर्ने र सहायता प्राप्त गर्ने  

यो परियोजनाले बग र सुविधा अनुरोधहरू ट्र्याक गर्न GitHub Issues प्रयोग गर्दछ। कृपया नयाँ मुद्दा दर्ता गर्नु अघि पहिलेका मुद्दाहरू खोज्नुहोस् ताकि दोहोर्याइबाट बच्न सकियोस्। नयाँ मुद्दाहरूका लागि, आफ्नो बग वा सुविधा अनुरोध नयाँ Issue को रूपमा दर्ता गर्नुहोस्।

## Microsoft समर्थन नीति  

यस परियोजनाको समर्थन माथि सूचीबद्ध स्रोतहरूमा सीमित छ।
