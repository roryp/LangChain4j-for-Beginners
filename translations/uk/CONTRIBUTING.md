# Contributing

Дякуємо за вашу зацікавленість у внесенні змін! Ми вітаємо pull
запити та покращення цього курсу. Будь ласка, дотримуйтесь цих
рекомендацій при поданні змін:

1. **Без секретів у комітах.** Ніколи не комітьте API-ключі або будь-які
   конфіденційні матеріали. Використовуйте локальний `.env` для власного тестування.
2. **Зберігайте фокус змін.** Невеликі, сфокусовані pull-запити легше
   переглядати та підтримувати. Якщо ви виправляєте кілька проблем, будь ласка,
   подавайте їх окремо.
3. **Додавайте тести, де можливо.** Юніт- або інтеграційні тести допомагають
   виявляти регресії. Для модулів Spring Boot включайте тести у папку
   `src/test/java`.
4. **Дотримуйтесь конвенцій стилю коду.** Використовуйте існуючу структуру проекту як орієнтир. Ми покладаємось на Maven для керування залежностями та
   плагін Spring Boot Maven для пакування додатків.

Дякуємо, що допомагаєте зробити цей курс кращим!
