# Підтримка

## Як подавати проблеми та отримувати допомогу  

Цей проєкт використовує GitHub Issues для відстеження помилок і запитів на функції. Будь ласка, пошукайте існуючі 
проблеми перед поданням нових, щоб уникнути дублювань. Для нових проблем подайте вашу помилку або 
запит на функцію як нову проблему.

## Політика підтримки Microsoft  

Підтримка цього проєкту обмежується ресурсами, переліченими вище.
