# LangChain4j untuk Permulaan

Satu kursus untuk membina aplikasi AI dengan LangChain4j dan Azure OpenAI GPT-5, daripada chat asas hingga ejen AI.

**Baru dengan LangChain4j?** Lihat [Glosari](docs/GLOSSARY.md) untuk definisi istilah dan konsep penting.

## Jadual Kandungan

1. [Permulaan Pantas](00-quick-start/README.md) - Mulakan dengan LangChain4j  
2. [Pengenalan](01-introduction/README.md) - Pelajari asas-asas LangChain4j  
3. [Kejuruteraan Prompt](02-prompt-engineering/README.md) - Kuasai reka bentuk prompt yang berkesan  
4. [RAG (Retrieval-Augmented Generation)](03-rag/README.md) - Bina sistem berasaskan pengetahuan yang pintar  
5. [Alat](04-tools/README.md) - Gabungkan alat luaran dan pembantu mudah  
6. [MCP (Protokol Konteks Model)](05-mcp/README.md) - Bekerja dengan Protokol Konteks Model (MCP) dan modul Agen  
---

## Jalur Pembelajaran

> **Permulaan Pantas**

1. Fork repositori ini ke akaun GitHub anda  
2. Klik **Code** ‚Üí tab **Codespaces** ‚Üí **...** ‚Üí **Baru dengan pilihan...**  
3. Gunakan tetapan lalai ‚Äì ini akan memilih bekas Pembangunan yang dibuat untuk kursus ini  
4. Klik **Cipta kodspace**  
5. Tunggu 5-10 minit untuk persekitaran bersedia  
6. Teruskan ke [Permulaan Pantas](./00-quick-start/README.md) untuk memulakan!

Setelah selesai modul, terokai [Panduan Pengujian](docs/TESTING.md) untuk melihat konsep pengujian LangChain4j dalam tindakan.

> **Nota:** Latihan ini menggunakan kedua-dua Model GitHub dan Azure OpenAI. Modul [Permulaan Pantas](00-quick-start/README.md) menggunakan Model GitHub (tidak memerlukan langganan Azure), manakala modul 1-5 menggunakan Azure OpenAI.

## Pembelajaran dengan GitHub Copilot

Untuk mula menulis kod dengan cepat, buka projek ini dalam GitHub Codespace atau IDE tempatan anda dengan devcontainer yang disediakan. Devcontainer yang digunakan dalam kursus ini sudah dikonfigurasikan dengan GitHub Copilot untuk pengaturcaraan berpasangan AI.

Setiap contoh kod termasuk soalan yang dicadangkan yang boleh anda tanya pada GitHub Copilot untuk mendalami pemahaman anda. Cari isyarat üí°/ü§ñ dalam:

- **Header fail Java** - Soalan khusus untuk setiap contoh  
- **README modul** - Cadangan penerokaan selepas contoh kod  

**Cara guna:** Buka mana-mana fail kod dan tanyakan soalan yang dicadangkan kepada Copilot. Ia mempunyai konteks penuh bagi kod dan boleh menerangkan, meluaskan, dan mencadangkan alternatif.

Mahukan lebih banyak maklumat? Lihat [Copilot untuk Pengaturcaraan Berpasangan AI](https://aka.ms/GitHubCopilotAI).
