# Menyumbang

Terima kasih atas minat anda untuk menyumbang! Kami mengalu-alukan permintaan tarik dan penambahbaikan untuk kursus ini. Sila ikuti garis panduan ini apabila menghantar perubahan:

1. **Tiada rahsia dalam komit.** Jangan sekali-kali komit kunci API atau sebarang bahan sulit. Gunakan `.env` tempatan untuk ujian anda sendiri.
2. **Pastikan perubahan fokus.** Permintaan tarik yang kecil dan terfokus lebih mudah untuk disemak dan diselenggara. Jika anda membaiki pelbagai isu, sila hantar secara berasingan.
3. **Tambah ujian jika boleh.** Ujian unit atau integrasi membantu mengesan regresi. Untuk modul Spring Boot, sertakan ujian di bawah folder `src/test/java`.
4. **Ikuti konvensyen gaya kod.** Gunakan struktur projek sedia ada sebagai panduan. Kami bergantung pada Maven untuk menguruskan kebergantungan dan plugin Spring Boot Maven untuk membungkus aplikasi.

Terima kasih kerana membantu menjadikan kursus ini lebih baik!
