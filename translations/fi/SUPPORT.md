# Tuki

## Kuinka tehdä ilmoituksia ja saada apua  

Tämä projekti käyttää GitHub Issues -järjestelmää virheiden ja ominaisuuspyyntöjen seuraamiseen. Etsi olemassa olevat 
ilmoitukset ennen uusien tekemistä, jotta vältät päällekkäisyyksiä. Uusissa ilmoituksissa tee virhe- tai 
ominaisuuspyyntö uutena Issue-ilmoituksena.

## Microsoftin tukipolitiikka  

Tämän projektin tuki rajoittuu yllä lueteltuihin resursseihin.
