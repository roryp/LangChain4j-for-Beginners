# Podpora

## Ako podať problémy a získať pomoc  

Tento projekt používa GitHub Issues na sledovanie chýb a požiadaviek na funkcie. Pred podaním nových problémov prosím vyhľadajte existujúce problémy, aby ste sa vyhli duplicitám. Pre nové problémy podajte svoju chybu alebo požiadavku na funkciu ako nový Issue.
