# Príspevky

Ďakujeme za váš záujem prispieť! Radi prijímame pull requesty a vylepšenia tohto kurzu. Pri zasielaní zmien prosím dodržiavajte tieto pokyny:

1. **Žiadne tajomstvá v commitoch.** Nikdy necommitujte API kľúče alebo akýkoľvek dôverný materiál. Pre vlastné testovanie používajte lokálny `.env`.
2. **Zamerajte sa na konkrétne zmeny.** Malé, zamerané pull requesty sa ľahšie kontrolujú a udržiavajú. Ak opravujete viacero problémov, pošlite ich prosím samostatne.
3. **Pridajte testy, kde je to možné.** Jednotkové alebo integračné testy pomáhajú zachytiť regresie. Pre moduly Spring Boot zahrňte testy do priečinka `src/test/java`.
4. **Dodržiavajte konvencie štýlu kódu.** Používajte existujúcu štruktúru projektu ako vodítko. Spoliehame sa na Maven na správu závislostí a na Spring Boot Maven plugin na balenie aplikácií.

Ďakujeme, že pomáhate zlepšiť tento kurz!
