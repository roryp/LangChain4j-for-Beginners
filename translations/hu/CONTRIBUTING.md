# Hozzájárulás

Köszönjük az érdeklődését a hozzájárulás iránt! Örömmel fogadjuk a pull kéréseket és a kurzus fejlesztéseit. Kérjük, kövesse az alábbi irányelveket a változtatások beküldésekor:

1. **Ne legyenek titkok a commitokban.** Soha ne commitáljon API kulcsokat vagy bármilyen bizalmas anyagot. Saját teszteléshez használjon helyi `.env` fájlt.
2. **Tartsa fókuszáltan a változtatásokat.** A kis, jól körülhatárolt pull kérések könnyebben áttekinthetők és karbantarthatók. Ha több problémát javít, kérjük, külön-külön küldje be őket.
3. **Adjon hozzá teszteket, ahol lehetséges.** Az egység- vagy integrációs tesztek segítenek a visszaesések elkerülésében. A Spring Boot modulok esetén helyezze a teszteket a `src/test/java` mappába.
4. **Kövesse a kódstílus konvenciókat.** Használja a meglévő projekt struktúrát útmutatóként. A függőségek kezeléséhez Maven-t, az alkalmazások csomagolásához pedig a Spring Boot Maven plugint használjuk.

Köszönjük, hogy segít jobbá tenni ezt a kurzust!
