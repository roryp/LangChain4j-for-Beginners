# Contributing

Dėkojame už susidomėjimą prisidėti! Mes laukiame pull request'ų ir patobulinimų šiam kursui. Prašome laikytis šių gairių pateikiant pakeitimus:

1. **Nėra paslapčių commit'uose.** Niekada neįtraukite API raktų ar jokios konfidencialios medžiagos. Naudokite vietinį `.env` savo testavimui.
2. **Laikykite pakeitimus susitelkusius.** Maži, apibrėžti pull request'ai yra lengviau peržiūrimi ir prižiūrimi. Jei taisote kelias problemas, pateikite jas atskirai.
3. **Pridėkite testus, jei įmanoma.** Vienetiniai arba integraciniai testai padeda aptikti regresijas. Spring Boot moduliams įtraukite testus į `src/test/java` katalogą.
4. **Laikykitės kodo stiliaus konvencijų.** Naudokite esamą projekto struktūrą kaip gairę. Mes naudojame Maven priklausomybėms valdyti ir Spring Boot Maven plugin'ą programų pakavimui.

Dėkojame, kad padedate tobulinti šį kursą!
