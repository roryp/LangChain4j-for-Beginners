## Saugumas

Microsoft rimtai žiūri į mūsų programinės įrangos produktų ir paslaugų saugumą, įskaitant visus šaltinio kodo saugyklas, valdomas per mūsų GitHub organizacijas, kurios apima [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) ir [Xamarin](https://github.com/xamarin).

Jei manote, kad radote saugumo spragą bet kurioje Microsoft priklausančioje saugykloje, atitinkančią [Microsoft saugumo spragos apibrėžimą](https://aka.ms/security.md/definition), prašome pranešti mums, kaip aprašyta žemiau.

## Saugumo problemų pranešimas

**Prašome neskelbti saugumo spragų per viešas GitHub problemas.**

Vietoje to, prašome pranešti Microsoft saugumo reagavimo centrui (MSRC) adresu [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Jei norite pateikti pranešimą be prisijungimo, siųskite el. laišką adresu [secure@microsoft.com](mailto:secure@microsoft.com). Jei įmanoma, užšifruokite savo žinutę mūsų PGP raktu; jį galite atsisiųsti iš [Microsoft saugumo reagavimo centro PGP rakto puslapio](https://aka.ms/security.md/msrc/pgp).

Turėtumėte gauti atsakymą per 24 valandas. Jei dėl kokios nors priežasties negaunate, prašome susisiekti el. paštu, kad įsitikintumėte, jog gavome jūsų pradinį pranešimą. Papildoma informacija pateikiama [microsoft.com/msrc](https://www.microsoft.com/msrc).

Prašome pateikti žemiau nurodytą informaciją (kiek įmanoma), kad padėtumėte mums geriau suprasti galimos problemos pobūdį ir mastą:

  * Problemos tipas (pvz., buferio perpildymas, SQL injekcija, kryžminis svetainių skriptavimas ir kt.)
  * Pilni šaltinio failų keliai, susiję su problemos pasireiškimu
  * Pažeisto šaltinio kodo vieta (žyma/šaka/įsipareigojimas arba tiesioginis URL)
  * Bet kokia speciali konfigūracija, reikalinga problemai atkurti
  * Žingsnis po žingsnio instrukcijos, kaip atkurti problemą
  * Įrodymo koncepcijos arba išnaudojimo kodas (jei įmanoma)
  * Problemos poveikis, įskaitant kaip užpuolikas galėtų išnaudoti problemą

Ši informacija padės mums greičiau apdoroti jūsų pranešimą.

Jei pranešate dėl klaidų atlygio, išsamesni pranešimai gali prisidėti prie didesnio atlygio. Daugiau informacijos rasite mūsų [Microsoft klaidų atlygio programos](https://aka.ms/security.md/msrc/bounty) puslapyje apie aktyvias programas.
