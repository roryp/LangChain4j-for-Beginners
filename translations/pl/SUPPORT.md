# Wsparcie

## Jak zgłaszać problemy i uzyskać pomoc  

Ten projekt korzysta z GitHub Issues do śledzenia błędów i próśb o funkcje. Proszę przeszukać istniejące 
zgłoszenia przed dodaniem nowych, aby uniknąć duplikatów. W przypadku nowych problemów zgłoś swój błąd lub 
prośbę o funkcję jako nowe zgłoszenie.
