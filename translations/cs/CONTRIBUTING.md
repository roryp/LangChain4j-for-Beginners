# Přispívání

Děkujeme za váš zájem o přispívání! Vítáme pull requesty a vylepšení tohoto kurzu. Při zasílání změn prosím dodržujte tyto pokyny:

1. **Žádná tajemství v commitech.** Nikdy necommitujte API klíče nebo jakýkoli důvěrný materiál. Pro vlastní testování používejte lokální `.env`.
2. **Udržujte změny zaměřené.** Malé, vymezené pull requesty se lépe kontrolují a udržují. Pokud opravujete více problémů, prosím zasílejte je zvlášť.
3. **Přidávejte testy, kde je to možné.** Jednotkové nebo integrační testy pomáhají odhalit regresi. Pro moduly Spring Boot zahrňte testy do složky `src/test/java`.
4. **Dodržujte konvence stylu kódu.** Použijte stávající strukturu projektu jako vodítko. Spoléháme na Maven pro správu závislostí a Spring Boot Maven plugin pro balení aplikací.

Děkujeme, že pomáháte zlepšovat tento kurz!
