# Podpora

## Jak podávat problémy a získat pomoc  

Tento projekt používá GitHub Issues pro sledování chyb a požadavků na funkce. Před podáním nových problémů prosím vyhledejte existující problémy, abyste se vyhnuli duplicitám. Pro nové problémy podávejte svou chybu nebo požadavek na funkci jako nový Issue.

## Podpůrná politika Microsoftu  

Podpora pro tento projekt je omezena na výše uvedené zdroje.
