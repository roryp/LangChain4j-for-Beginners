# Support

## Hvordan sende inn problemer og få hjelp  

Dette prosjektet bruker GitHub Issues for å spore feil og funksjonsforespørsler. Vennligst søk i eksisterende 
problemer før du sender inn nye for å unngå duplikater. For nye problemer, send inn feilen eller 
funksjonsforespørselen som en ny Issue.

## Microsofts støttepolicy  

Støtte for dette prosjektet er begrenset til ressursene som er oppført ovenfor.
