# Συμβολή

Σας ευχαριστούμε για το ενδιαφέρον σας να συμβάλετε! Καλωσορίζουμε pull requests και βελτιώσεις σε αυτό το μάθημα. Παρακαλούμε ακολουθήστε αυτές τις οδηγίες κατά την υποβολή αλλαγών:

1. **Μη βάζετε μυστικά στα commits.** Ποτέ μην κάνετε commit κλειδιά API ή οποιοδήποτε εμπιστευτικό υλικό. Χρησιμοποιήστε ένα τοπικό `.env` για τις δικές σας δοκιμές.
2. **Κρατήστε τις αλλαγές εστιασμένες.** Μικρά, στοχευμένα pull requests είναι πιο εύκολα στην ανασκόπηση και τη συντήρηση. Αν διορθώνετε πολλαπλά ζητήματα, παρακαλούμε υποβάλετέ τα ξεχωριστά.
3. **Προσθέστε δοκιμές όπου είναι δυνατόν.** Οι μονάδες ή οι ολοκληρωμένες δοκιμές βοηθούν στην ανίχνευση παλινδρομήσεων. Για τα modules του Spring Boot, συμπεριλάβετε δοκιμές στον φάκελο `src/test/java`.
4. **Ακολουθήστε τις συμβάσεις στυλ κώδικα.** Χρησιμοποιήστε τη υπάρχουσα δομή του έργου ως οδηγό. Βασιζόμαστε στο Maven για τη διαχείριση εξαρτήσεων και στο Spring Boot Maven plugin για τη συσκευασία των εφαρμογών.

Σας ευχαριστούμε που βοηθάτε να γίνει αυτό το μάθημα καλύτερο!
