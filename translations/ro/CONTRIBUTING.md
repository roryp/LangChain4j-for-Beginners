# Contribuții

Vă mulțumim pentru interesul de a contribui! Primim cu plăcere pull request-uri și îmbunătățiri pentru acest curs. Vă rugăm să urmați aceste linii directoare când trimiteți modificări:

1. **Fără secrete în commit-uri.** Nu faceți niciodată commit-uri cu chei API sau materiale confidențiale. Folosiți un fișier local `.env` pentru testarea proprie.
2. **Păstrați modificările concentrate.** Pull request-urile mici și bine delimitate sunt mai ușor de revizuit și întreținut. Dacă remediați mai multe probleme, vă rugăm să le trimiteți separat.
3. **Adăugați teste acolo unde este posibil.** Testele unitare sau de integrare ajută la prinderea regresiilor. Pentru modulele Spring Boot, includeți teste în folderul `src/test/java`.
4. **Respectați convențiile de stil de cod.** Folosiți structura existentă a proiectului ca ghid. Ne bazăm pe Maven pentru gestionarea dependențelor și pe plugin-ul Spring Boot Maven pentru împachetarea aplicațiilor.

Vă mulțumim că ajutați la îmbunătățirea acestui curs!
