# Support

## Comment signaler des problèmes et obtenir de l'aide  

Ce projet utilise GitHub Issues pour suivre les bugs et les demandes de fonctionnalités. Veuillez rechercher les problèmes existants avant d'en signaler de nouveaux afin d'éviter les doublons. Pour les nouveaux problèmes, déposez votre bug ou demande de fonctionnalité en tant que nouveau problème.
