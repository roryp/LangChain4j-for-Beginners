# Suporta

## Paano mag-file ng mga isyu at humingi ng tulong  

Gumagamit ang proyektong ito ng GitHub Issues upang subaybayan ang mga bug at kahilingan para sa mga tampok. Mangyaring hanapin muna ang mga umiiral na isyu bago mag-file ng mga bagong isyu upang maiwasan ang mga duplicate. Para sa mga bagong isyu, i-file ang iyong bug o kahilingan para sa tampok bilang bagong Isyu.

## Patakaran sa Suporta ng Microsoft  

Ang suporta para sa proyektong ito ay limitado sa mga mapagkukunan na nakalista sa itaas.
