# Contributing

Salamat sa iyong interes na mag-ambag! Tinatanggap namin ang mga pull request at mga pagpapabuti sa kursong ito. Mangyaring sundin ang mga alituntuning ito kapag nagsusumite ng mga pagbabago:

1. **Walang mga lihim sa mga commit.** Huwag kailanman mag-commit ng mga API key o anumang kumpidensyal na materyal. Gumamit ng lokal na `.env` para sa iyong sariling pagsubok.
2. **Panatilihing nakatuon ang mga pagbabago.** Mas madali suriin at panatilihin ang maliliit at nakatuong pull request. Kung inaayos mo ang maraming isyu, mangyaring isumite ang mga ito nang hiwalay.
3. **Magdagdag ng mga pagsubok kung maaari.** Nakakatulong ang mga unit o integration test upang mahuli ang mga regression. Para sa mga Spring Boot module, isama ang mga pagsubok sa ilalim ng `src/test/java` na folder.
4. **Sundin ang mga konbensyon sa estilo ng code.** Gamitin ang umiiral na istruktura ng proyekto bilang gabay. Umaasa kami sa Maven upang pamahalaan ang mga dependency at sa Spring Boot Maven plugin upang i-package ang mga aplikasyon.

Salamat sa pagtulong na pagbutihin ang kursong ito!
