# LangChain4j kwa Waanzaji

Kozi ya kujenga programu za AI na LangChain4j na Azure OpenAI GPT-5, kuanzia mazungumzo ya msingi hadi maajenti wa AI.

**Mpya kwa LangChain4j?** Angalia [Glosari](docs/GLOSSARY.md) kwa ufafanuzi wa maneno na dhana muhimu.

## Jedwali la Yaliyomo

1. [Anza Haraka](00-quick-start/README.md) - Anza na LangChain4j
2. [Utangulizi](01-introduction/README.md) - Jifunze misingi ya LangChain4j
3. [Uhandisi wa Amri (Prompt Engineering)](02-prompt-engineering/README.md) - Jifunze muundo mzuri wa amri
4. [RAG (Uzalishaji uliosaidiwa na Utafutaji)](03-rag/README.md) - Jenga mifumo ya akili yenye maarifa
5. [Zana](04-tools/README.md) - Unganisha zana za nje na wasaidizi rahisi
6. [MCP (Itifaki ya Muktadha wa Mfano)](05-mcp/README.md) - Fanya kazi na Itifaki ya Muktadha wa Mfano (MCP) na moduli za Machaguo
---

## Njia ya Kujifunza

> **Anza Haraka**

1. Tengeneza nakala ya hifadhidata hii kwenye akaunti yako ya GitHub
2. Bofya **Msimbo** ‚Üí kichupo cha **Codespaces** ‚Üí **...** ‚Üí **Mpya na chaguzi...**
3. Tumia chaguo za msingi ‚Äì hii itachagua kontena ya Maendeleo iliyotengenezwa kwa ajili ya kozi hii
4. Bofya **Tengeneza codespace**
5. Subiri dakika 5-10 ili mazingira yawe tayari
6. Nenda moja kwa moja kwa [Anza Haraka](./00-quick-start/README.md) kuanza!

> **Ungependa Kwenye Nakili Moja kwa Moja?**
>
> Hifadhidata hii ina tafsiri za lugha zaidi ya 50 ambazo huongeza kwa kiasi kikubwa ukubwa wa kupakua. Ili kunakili bila tafsiri, tumia sparse checkout:
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/LangChain4j-for-Beginners.git
> cd LangChain4j-for-Beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> Hii inakupa kila kitu unachohitaji kukamilisha kozi kwa upakuaji wa kasi zaidi.

Baada ya kumaliza moduli, chunguza [Mwongozo wa Upimaji](docs/TESTING.md) kuona dhana za upimaji za LangChain4j katika vitendo.

> **Kumbuka:** Mafunzo haya yanatumia Mifano ya GitHub na Azure OpenAI. Moduli ya [Anza Haraka](00-quick-start/README.md) inatumia Mifano ya GitHub (hakuna usajili wa Azure unaohitajika), wakati moduli 1-5 zinatumia Azure OpenAI.


## Kujifunza na GitHub Copilot

Ili kuanza kuandika msimbo kwa haraka, fungua mradi huu kwenye GitHub Codespace au IDE yako ya eneo-kazi kwa kutumia devcontainer iliyotolewa. Devcontainer inayotumiwa kwenye kozi hii imewekwa awali na GitHub Copilot kwa ajili ya programu ya kuandikisha kwa upana wa AI.

Kila mfano wa msimbo una maswali yaliyo pendekezwa unayoweza kumuuliza GitHub Copilot ili kuongeza uelewa wako. Tafuta vihimizo vya üí°/ü§ñ katika:

- **Vichwa vya faili za Java** - Maswali maalum kwa mfano wowote
- **README za Moduli** - Vihimizo vya uchunguzi baada ya mifano ya msimbo

**Jinsi ya kutumia:** Fungua faili lolote la msimbo na uliza maswali yaliyopendekezwa kwa Copilot. Ina muktadha kamili wa msimbo na inaweza kufafanua, kuongeza, na kupendekeza mbadala.

Unataka kujifunza zaidi? Angalia [Copilot kwa Kuprogramu kwa Pamoja na AI](https://aka.ms/GitHubCopilotAI).
