# ਸਹਾਇਤਾ

## ਮੁੱਦੇ ਦਰਜ ਕਰਨ ਅਤੇ ਮਦਦ ਪ੍ਰਾਪਤ ਕਰਨ ਦਾ ਤਰੀਕਾ  

ਇਹ ਪ੍ਰੋਜੈਕਟ ਬੱਗ ਅਤੇ ਫੀਚਰ ਬੇਨਤੀਆਂ ਨੂੰ ਟਰੈਕ ਕਰਨ ਲਈ GitHub Issues ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ। ਕਿਰਪਾ ਕਰਕੇ ਨਵੇਂ ਮੁੱਦੇ ਦਰਜ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਮੌਜੂਦਾ ਮੁੱਦਿਆਂ ਦੀ ਖੋਜ ਕਰੋ ਤਾਂ ਜੋ ਨਕਲਾਂ ਤੋਂ ਬਚਿਆ ਜਾ ਸਕੇ। ਨਵੇਂ ਮੁੱਦਿਆਂ ਲਈ, ਆਪਣਾ ਬੱਗ ਜਾਂ ਫੀਚਰ ਬੇਨਤੀ ਇੱਕ ਨਵੇਂ Issue ਵਜੋਂ ਦਰਜ ਕਰੋ।

## ਮਾਈਕ੍ਰੋਸਾਫਟ ਸਹਾਇਤਾ ਨੀਤੀ
