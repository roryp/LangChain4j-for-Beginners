# সমর্থন

## কীভাবে সমস্যা দায়ের করবেন এবং সাহায্য পাবেন  

এই প্রকল্পটি বাগ এবং ফিচার অনুরোধ ট্র্যাক করার জন্য GitHub Issues ব্যবহার করে। অনুগ্রহ করে নতুন সমস্যা দায়ের করার আগে বিদ্যমান সমস্যাগুলি অনুসন্ধান করুন যাতে ডুপ্লিকেট এড়ানো যায়। নতুন সমস্যার জন্য, আপনার বাগ বা ফিচার অনুরোধ একটি নতুন Issue হিসাবে দায়ের করুন।

## Microsoft সমর্থন নীতি  

এই প্রকল্পের জন্য সমর্থন উপরে তালিকাভুক্ত সম্পদগুলির মধ্যে সীমাবদ্ধ।
