# Suporte

## Como registrar problemas e obter ajuda  

Este projeto usa o GitHub Issues para rastrear bugs e solicitações de recursos. Por favor, pesquise os problemas existentes antes de registrar novos problemas para evitar duplicatas. Para novos problemas, registre seu bug ou solicitação de recurso como um novo Issue.

## Política de Suporte da Microsoft  

O suporte para este projeto é limitado aos recursos listados acima.
