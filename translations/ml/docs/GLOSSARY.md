## പ്രധാന ആശയങ്ങൾ

**AI ഏജന്റ്** - സ്വയം ചിന്തിച്ച് പ്രവർത്തിക്കാൻ AI ഉപയോഗിക്കുന്ന സിസ്റ്റം. [Module 04](../04-tools/README.md)

**ചെയിൻ** - ഔട്ട്പുട്ട് അടുത്ത ഘട്ടത്തിലേക്ക് നൽകുന്ന പ്രവർത്തനങ്ങളുടെ ശ്രേണി.

**ചങ്കിംഗ്** - ഡോക്യുമെന്റുകൾ ചെറിയ ഭാഗങ്ങളായി വിഭജിക്കൽ. സാധാരണ: 300-500 ടോക്കൺസ് ഒവർലാപ്പോടെ. [Module 03](../03-rag/README.md)

**കോൺടെക്സ്റ്റ് വിൻഡോ** - മോഡൽ പ്രോസസ്സ് ചെയ്യാൻ കഴിയുന്ന പരമാവധി ടോക്കൺസ്. GPT-5: 400K ടോക്കൺസ്.

**എംബെഡ്ഡിംഗ്സ്** - വാചകത്തിന്റെ അർത്ഥം പ്രതിനിധീകരിക്കുന്ന സംഖ്യാത്മക വെക്ടറുകൾ. [Module 03](../03-rag/README.md)

**ഫംഗ്ഷൻ കോളിംഗ്** - മോഡൽ ബാഹ്യ ഫംഗ്ഷനുകൾ വിളിക്കാൻ ഘടനാപരമായ അഭ്യർത്ഥനകൾ സൃഷ്ടിക്കുന്നു. [Module 04](../04-tools/README.md)

**ഹല്യൂസിനേഷൻ** - മോഡലുകൾ തെറ്റായ പക്ഷേ വിശ്വസനീയമായ വിവരങ്ങൾ സൃഷ്ടിക്കുമ്പോൾ.

**പ്രോംപ്റ്റ്** - ഭാഷാ മോഡലിലേക്ക് നൽകുന്ന ടെക്സ്റ്റ് ഇൻപുട്ട്. [Module 02](../02-prompt-engineering/README.md)

**സെമാന്റിക് സെർച്ച്** - കീവേഡുകൾക്കുപകരം അർത്ഥം ഉപയോഗിച്ച് തിരയൽ. [Module 03](../03-rag/README.md)

**സ്റ്റേറ്റ്‌ഫുൾ vs സ്റ്റേറ്റ്‌ലെസ്** - സ്റ്റേറ്റ്‌ലെസ്: ഓർമ്മയില്ല. സ്റ്റേറ്റ്‌ഫുൾ: സംഭാഷണ ചരിത്രം സൂക്ഷിക്കുന്നു. [Module 01](../01-introduction/README.md)

**ടോക്കൺസ്** - മോഡലുകൾ പ്രോസസ്സ് ചെയ്യുന്ന അടിസ്ഥാന ടെക്സ്റ്റ് യൂണിറ്റുകൾ. ചെലവുകളും പരിധികളും ബാധിക്കുന്നു. [Module 01](../01-introduction/README.md)

**ടൂൾ ചെയിനിംഗ്** - ഔട്ട്പുട്ട് അടുത്ത വിളിക്കായി ഉപയോഗിക്കുന്ന ടൂളുകളുടെ പരമ്പര.

## LangChain4j ഘടകങ്ങൾ

**AiServices** - ടൈപ്പ്-സേഫ് AI സർവീസ് ഇന്റർഫേസുകൾ സൃഷ്ടിക്കുന്നു.

**OpenAiOfficialChatModel** - OpenAI, അസ്യൂർ OpenAI മോഡലുകൾക്കുള്ള ഏകീകൃത ക്ലയന്റ്.

**OpenAiOfficialEmbeddingModel** - OpenAI ഔദ്യോഗിക ക്ലയന്റ് ഉപയോഗിച്ച് എംബെഡ്ഡിംഗുകൾ സൃഷ്ടിക്കുന്നു (OpenAI, അസ്യൂർ OpenAI രണ്ടും പിന്തുണയ്ക്കുന്നു).

**ChatModel** - ഭാഷാ മോഡലുകളുടെ കോർ ഇന്റർഫേസ്.

**ChatMemory** - സംഭാഷണ ചരിത്രം സൂക്ഷിക്കുന്നു.

**ContentRetriever** - RAG-ക്കായി ബന്ധപ്പെട്ട ഡോക്യുമെന്റ് ചങ്കുകൾ കണ്ടെത്തുന്നു.

**DocumentSplitter** - ഡോക്യുമെന്റുകൾ ചങ്കുകളായി വിഭജിക്കുന്നു.

**EmbeddingModel** - ടെക്സ്റ്റ് സംഖ്യാത്മക വെക്ടറുകളായി മാറ്റുന്നു.

**EmbeddingStore** - എംബെഡ്ഡിംഗുകൾ സൂക്ഷിക്കുകയും തിരികെ ലഭ്യമാക്കുകയും ചെയ്യുന്നു.

**MessageWindowChatMemory** - അടുത്തിടെ വന്ന സന്ദേശങ്ങളുടെ സ്ലൈഡിംഗ് വിൻഡോ സൂക്ഷിക്കുന്നു.

**PromptTemplate** - `{{variable}}` പ്ലേസ്ഹോൾഡറുകൾ ഉപയോഗിച്ച് പുനരുപയോഗയോഗ്യമായ പ്രോംപ്റ്റുകൾ സൃഷ്ടിക്കുന്നു.

**TextSegment** - മെറ്റാഡേറ്റയോടുകൂടിയ ടെക്സ്റ്റ് ചങ്ക്. RAG-യിൽ ഉപയോഗിക്കുന്നു.

**ToolExecutionRequest** - ടൂൾ എക്സിക്യൂഷൻ അഭ്യർത്ഥന പ്രതിനിധാനം ചെയ്യുന്നു.

**UserMessage / AiMessage / SystemMessage** - സംഭാഷണ സന്ദേശ തരം.

## AI/ML ആശയങ്ങൾ

**Few-Shot Learning** - പ്രോംപ്റ്റുകളിൽ ഉദാഹരണങ്ങൾ നൽകൽ. [Module 02](../02-prompt-engineering/README.md)

**ലാർജ് ലാംഗ്വേജ് മോഡൽ (LLM)** - വൻ ടെക്സ്റ്റ് ഡാറ്റയിൽ പരിശീലിപ്പിച്ച AI മോഡലുകൾ.

**റീസണിംഗ് എഫോർട്ട്** - ചിന്തനത്തിന്റെ ആഴം നിയന്ത്രിക്കുന്ന GPT-5 പാരാമീറ്റർ. [Module 02](../02-prompt-engineering/README.md)

**ടെംപറേച്ചർ** - ഔട്ട്പുട്ടിന്റെ അനിശ്ചിതത്വം നിയന്ത്രിക്കുന്നു. കുറവ്=നിശ്ചിതം, ഉയരം=സൃഷ്ടിപരമായ.

**വെക്ടർ ഡാറ്റാബേസ്** - എംബെഡ്ഡിംഗുകൾക്കായി പ്രത്യേകിച്ചുള്ള ഡാറ്റാബേസ്. [Module 03](../03-rag/README.md)

**സീറോ-ഷോട്ട് ലേണിംഗ്** - ഉദാഹരണങ്ങളില്ലാതെ പ്രവർത്തനങ്ങൾ നടത്തൽ. [Module 02](../02-prompt-engineering/README.md)

## പ്രോംപ്റ്റ് എഞ്ചിനീയറിംഗ് - [Module 02](../02-prompt-engineering/README.md)

**ചെയിൻ-ഓഫ്-തോട്ട്** - മെച്ചപ്പെട്ട കൃത്യതയ്ക്കായി ഘട്ടം ഘട്ടമായ ചിന്തനം.

**കൺസ്ട്രെയിൻഡ് ഔട്ട്പുട്ട്** - പ്രത്യേക ഫോർമാറ്റ് അല്ലെങ്കിൽ ഘടന നിർബന്ധിക്കുക.

**ഹൈ ഈഗർനെസ്** - സമഗ്രമായ ചിന്തനത്തിനുള്ള GPT-5 പാറ്റേൺ.

**ലോ ഈഗർനെസ്** - വേഗത്തിലുള്ള ഉത്തരങ്ങൾക്ക് GPT-5 പാറ്റേൺ.

**മൾട്ടി-ടേൺ സംഭാഷണം** - എക്സ്ചേഞ്ചുകൾക്കിടയിൽ കോൺടെക്സ്റ്റ് നിലനിർത്തൽ.

**റോൾ-ബേസ്ഡ് പ്രോംപ്റ്റിംഗ്** - സിസ്റ്റം സന്ദേശങ്ങളിലൂടെ മോഡൽ വ്യക്തിത്വം സജ്ജമാക്കൽ.

**സെൽഫ്-റിഫ്ലെക്ഷൻ** - മോഡൽ തന്റെ ഔട്ട്പുട്ട് വിലയിരുത്തുകയും മെച്ചപ്പെടുത്തുകയും ചെയ്യുന്നു.

**സ്ട്രക്ചർഡ് അനാലിസിസ്** - സ്ഥിരമായ മൂല്യനിർണയ ഫ്രെയിംവർക്ക്.

**ടാസ്‌ക് എക്സിക്യൂഷൻ പാറ്റേൺ** - പ്ലാൻ → എക്സിക്യൂട്ട് → സംഗ്രഹിക്കുക.

## RAG (റിട്രീവൽ-ഓഗ്മെന്റഡ് ജനറേഷൻ) - [Module 03](../03-rag/README.md)

**ഡോക്യുമെന്റ് പ്രോസസ്സിംഗ് പൈപ്പ്‌ലൈൻ** - ലോഡ് → ചങ്ക് → എംബെഡ് → സ്റ്റോർ.

**ഇൻ-മെമ്മറി എംബെഡ്ഡിംഗ് സ്റ്റോർ** - ടെസ്റ്റിംഗിനായി സ്ഥിരതയില്ലാത്ത സംഭരണം.

**RAG** - റിട്രീവലും ജനറേഷനും സംയോജിപ്പിച്ച് മറുപടികൾ ഉറപ്പാക്കുന്നു.

**സിമിലാരിറ്റി സ്കോർ** - സെമാന്റിക് സമാനതയുടെ അളവ് (0-1).

**സോഴ്‌സ് റഫറൻസ്** - റിട്രീവുചെയ്ത ഉള്ളടക്കത്തെക്കുറിച്ചുള്ള മെറ്റാഡേറ്റ.

## ഏജന്റുകളും ടൂളുകളും - [Module 04](../04-tools/README.md)

**@Tool അനോട്ടേഷൻ** - ജാവ മെത്തഡുകൾ AI-കോളബിള്‍ ടൂളുകളായി അടയാളപ്പെടുത്തുന്നു.

**ReAct പാറ്റേൺ** - ചിന്തിക്കുക → പ്രവർത്തിക്കുക → നിരീക്ഷിക്കുക → ആവർത്തിക്കുക.

**സെഷൻ മാനേജ്മെന്റ്** - വ്യത്യസ്ത ഉപയോക്താക്കൾക്കായി വേർതിരിച്ച കോൺടെക്സ്റ്റുകൾ.

**ടൂൾ** - AI ഏജന്റ് വിളിക്കാൻ കഴിയുന്ന ഫംഗ്ഷൻ.

**ടൂൾ വിവരണം** - ടൂളിന്റെ ഉദ്ദേശവും പാരാമീറ്ററുകളും രേഖപ്പെടുത്തൽ.

## മോഡൽ കോൺടെക്സ്റ്റ് പ്രോട്ടോക്കോൾ (MCP) - [Module 05](../05-mcp/README.md)

**ഡോക്കർ ട്രാൻസ്പോർട്ട്** - ഡോക്കർ കണ്ടെയ്‌നറിൽ MCP സർവർ.

**MCP** - AI ആപ്പുകൾ ബാഹ്യ ടൂളുകളുമായി ബന്ധിപ്പിക്കുന്ന സ്റ്റാൻഡേർഡ്.

**MCP ക്ലയന്റ്** - MCP സർവറുകളുമായി ബന്ധിപ്പിക്കുന്ന ആപ്പ്.

**MCP സർവർ** - MCP വഴി ടൂളുകൾ പ്രദർശിപ്പിക്കുന്ന സർവീസ്.

**സർവർ-സെന്റ് ഇവന്റ്സ് (SSE)** - HTTP വഴി സർവർ-ടു-ക്ലയന്റ് സ്ട്രീമിംഗ്.

**സ്റ്റ്ഡിയോ ട്രാൻസ്പോർട്ട്** - stdin/stdout വഴി subprocess ആയി സർവർ.

**സ്ട്രീമബിൾ HTTP ട്രാൻസ്പോർട്ട്** - റിയൽ-ടൈം കമ്മ്യൂണിക്കേഷനായി SSE ഉള്ള HTTP.

**ടൂൾ ഡിസ്കവറി** - ലഭ്യമായ ടൂളുകൾക്കായി ക്ലയന്റ് സർവറിനെ ചോദ്യം ചെയ്യുന്നു.

## അസ്യൂർ സേവനങ്ങൾ - [Module 01](../01-introduction/README.md)

**അസ്യൂർ AI സെർച്ച്** - വെക്ടർ ശേഷിയുള്ള ക്ലൗഡ് സെർച്ച്. [Module 03](../03-rag/README.md)

**അസ്യൂർ ഡെവലപ്പർ CLI (azd)** - അസ്യൂർ റിസോഴ്‌സുകൾ ഡിപ്ലോയ് ചെയ്യുന്നു.

**അസ്യൂർ OpenAI** - മൈക്രോസോഫ്റ്റിന്റെ എന്റർപ്രൈസ് AI സേവനം.

**ബൈസപ്** - അസ്യൂർ ഇൻഫ്രാസ്ട്രക്ചർ-ആസ്-കോഡ് ഭാഷ. [Infrastructure Guide](../01-introduction/infra/README.md)

**ഡിപ്ലോയ്മെന്റ് നെയിം** - അസ്യൂറിൽ മോഡൽ ഡിപ്ലോയ്മെന്റിന് നാമം.

**GPT-5** - റീസണിംഗ് നിയന്ത്രണമുള്ള ഏറ്റവും പുതിയ OpenAI മോഡൽ. [Module 02](../02-prompt-engineering/README.md)
