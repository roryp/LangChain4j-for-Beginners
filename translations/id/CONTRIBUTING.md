# Contributing

Terima kasih atas minat Anda untuk berkontribusi! Kami menyambut pull request dan perbaikan untuk kursus ini. Harap ikuti panduan ini saat mengirimkan perubahan:

1. **Jangan ada rahasia dalam commit.** Jangan pernah meng-commit kunci API atau materi rahasia apa pun. Gunakan `.env` lokal untuk pengujian Anda sendiri.
2. **Jaga perubahan tetap fokus.** Pull request yang kecil dan terfokus lebih mudah untuk ditinjau dan dipelihara. Jika Anda memperbaiki beberapa masalah, harap kirimkan secara terpisah.
3. **Tambahkan tes jika memungkinkan.** Tes unit atau integrasi membantu menangkap regresi. Untuk modul Spring Boot, sertakan tes di bawah folder `src/test/java`.
4. **Ikuti konvensi gaya kode.** Gunakan struktur proyek yang ada sebagai panduan. Kami mengandalkan Maven untuk mengelola dependensi dan plugin Spring Boot Maven untuk mengemas aplikasi.

Terima kasih telah membantu membuat kursus ini menjadi lebih baik!
