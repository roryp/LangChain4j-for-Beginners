# การสนับสนุน

## วิธีการแจ้งปัญหาและขอความช่วยเหลือ  

โครงการนี้ใช้ GitHub Issues เพื่อติดตามข้อบกพร่องและคำขอคุณลักษณะ โปรดค้นหาปัญหาที่มีอยู่ก่อนที่จะแจ้งปัญหาใหม่เพื่อหลีกเลี่ยงการซ้ำซ้อน สำหรับปัญหาใหม่ ให้แจ้งข้อบกพร่องหรือคำขอคุณลักษณะของคุณเป็น Issue ใหม่

## นโยบายการสนับสนุนของ Microsoft  

การสนับสนุนสำหรับโครงการนี้จำกัดเฉพาะทรัพยากรที่ระบุไว้ข้างต้นเท่านั้น
